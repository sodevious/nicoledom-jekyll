I learned about the interactive Python console yesterday. It's pretty cool. 

To get inside, open Terminal.app and do this:
{% highlight python %}
$ python
{% endhighlight %}

To leave the console, 

{% highlight python %}
>>> quit()
{% endhighlight %}

You can do math, and print things too!
{% highlight python %}
>>> 1 + 1
2

>>> print "Hello World"
Hello World
{% endhighlight %}
